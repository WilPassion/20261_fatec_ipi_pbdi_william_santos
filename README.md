### 🎯 Objetivo

O principal objetivo desta disciplina é compreender e aplicar técnicas de programação para bancos de dados relacionais, com foco em:

* Manipulação e análise de dados.
* Automação de regras de negócio.
* Integridade e consistência dos dados.
* Controle de transações e concorrência.

### 🛠️ Tecnologias Utilizadas

* SQL.
* PL/SQL.
* Bancos de dados relacionais.
